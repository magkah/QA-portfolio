# API Testing Project (Postman)

This repository contains a Postman collection demonstrating API testing skills.

## 🔍 Scope
Collection includes tests for:
- JSONPlaceholder API (users, posts, comments)
- Basic authentication scenarios
- CRUD operations (GET, POST, PUT, DELETE)

## 🧪 Example test scenarios
- GET users → validate response structure
- POST request → verify payload and response
- PUT update → check data modification
- DELETE → verify successful response

## ⚙️ Features
- Environment variables (baseUrl, dynamic data)
- Automated test scripts (pm.test)
- Request chaining

## ▶️ How to use
1. Import `collection.json` into Postman
2. Run requests manually or using Collection Runner

## 💡 Skills demonstrated
- API testing (REST)
- Test design and validation
